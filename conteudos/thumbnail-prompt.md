# Prompt para Thumbnail — Natty or Not com IA

## Ferramenta recomendada
Midjourney, DALL·E 3 ou Stable Diffusion XL

## Prompt

```
A split-screen composition showing two contrasting sides: left side features a human content creator with natural lighting, realistic textures, working on a laptop in a modern minimalist workspace; right side shows an AI-generated holographic avatar creating content with glowing neural network patterns and futuristic interface elements. Both sides mirror each other in composition. Color palette: left side uses warm natural tones (beige, wood, soft daylight), right side uses electric blues, purples, and neon accents. Central dividing line with subtle glitch effect. Style: high-quality digital art, professional tech editorial, cinematic lighting, 16:9 aspect ratio. Text overlay space at top third. Clean, aesthetic, modern design. --ar 16:9 --style raw --v 6
```

## Variações de prompt (alternativas)

### Opção 2 — Minimalista
```
Minimalist tech illustration, split composition: left shows hand-drawn sketch style, right shows AI-generated perfect vector art. Gradient background from cream to electric blue. Central question mark symbol with glitch effect. Modern, clean, high contrast. --ar 16:9 --style raw
```

### Opção 3 — Conceitual
```
Abstract representation of human vs AI creativity: organic paint brushstrokes transforming into perfect geometric digital patterns, seamless transition in the middle, vibrant colors (coral, teal, gold), professional design poster style, balanced composition. --ar 16:9 --q 2
```

## Ajustes pós-geração
- Adicione texto com ferramenta de design (Canva/Figma):
  - Título: "NATTY OR NOT?"
  - Subtítulo: "O Guia Definitivo de IA Generativa"
  - Selo: "#LabDIONattyOrNot"
- Fonte: sans-serif moderna (Inter, Poppins, Montserrat)
- Tamanho recomendado final: 1920×1080px (Full HD)

## Dicas
- Use `--seed <número>` para gerar variações consistentes
- Para Midjourney: adicione `--v 6` para a versão mais recente
- Para DALL·E: especifique "no text in the image" se quiser adicionar texto depois
- Teste 3-5 variações e escolha a mais impactante
